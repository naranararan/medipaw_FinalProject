# medipaw
파이널 프로젝트

담당모듈: 병원관계자(security), 병원리뷰, 분양게시판 및 댓글

병원관계자 회원가입과 로그인은 일반회원과 구분되어있음
회원가입 시 사업자등록번호 상태조회(국세청 api)를 하여 국세청에 등록된 번호인 경우에만 회원가입이 가능하다.
